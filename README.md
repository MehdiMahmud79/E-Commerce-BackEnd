##  Installation
```bash
npm i
```

### this will install the following dependencies 
```bash
  "dependencies": {
    "dotenv": "^10.0.0",
    "express": "^4.17.1",
    "mysql2": "^2.1.0",
    "nodemon": "^2.0.12",
    "sequelize": "^5.21.7"
  }

```

## Run the app

  To use this app, you may need to run the following commands sequencially,

```bash
mysql -u root -p
=> source db/schema.sql;
quit

npm run dev
or
npm run start
```
